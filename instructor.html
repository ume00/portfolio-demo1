インストラクター紹介

それぞれの個性と専門性を兼ね備えた経験豊富なインストラクター陣が在籍しています。

初心者から上級者まで、様々なニーズとレベルに合わせて多彩なクラスを展開しております。

ヨガの智恵を活かし、皆さまの心身が理想の調和状態に至れるよう、私たちインストラクター一同でサポートさせていただきます。

Natsu

Profile

アロマセラピストの資格も持つNatsuは、香りの力を活用したリラックスヨガを得意とします。
ホットヨガでは太陽のエネルギーを、アロマヨガではアロマの香りを取り入れ、五感に働きかける体験を提供します。

Message

ヨガは魂の休息です。アロマの香りに導かれ、本当の自分に出会う旅に出てみましょう。

Class

アロマヨガ、ホットヨガ、ビギナークラス

戻る

Haruna

Profile

体と心の調和を生み出す、ピラティスの真髄を体現するのがHaruna。
ピラティスのエキスパートとして、効果的な動作とフォームを丁寧に指導し、姿勢の改善や体幹を鍛えることで、心身ともに健やかでバランスの取れた状態を作り出します。
静かな呼吸に意識を向けながら、ゆったりと身体を動かすことで、凝り固まった体の奥底から開放されていく感覚に浸れます。動きの一つひとつに意味があり、無理なく実践できるので、疲れ知らずの上質な時間が過ごせます。

Message

心地よい動きの中に、ひとつひとつ美しい姿勢と心の安らぎが宿っています。そんな小さな発見に出会える瞬間をお過ごしいただければと願っております。呼吸に意識を向け、穏やかなピラティスの時間を味わってください。

Class

ピラティス、マタニティヨガ、産後ヨガ

戻る

Hana

Profile

太陽礼拝に定評のある、Hanaの激しいパワフルなクラスは大人気です。
ヴィンヤサスタイルの動きを取り入れ、呼吸に合わせてアーサナを次々と展開していきます。
短時間で全身の筋肉をしっかりと使うため、心地よい疲労感が味わえると共に、肉体的・精神的なリフレッシュ効果が得られます。
ひたすらに動き続ける中で、気持ちが解放され、集中力が養われていきます。

Message

クラスが終わった後の、充実感と解放感は何物にも代えがたい体験です。一緒に熱い汗を流し、呼吸とアーサナの連動に夢中になれば、必ずや新しい自分に出会えることでしょう。ぜひ一緒に、夢心地のひと時をお過ごしください。

Class

ホットヨガ、アシュタンガヨガ、ビギナークラス

戻る

Satsuki

Profile

真摯で丁寧な指導が評判のSatsuki。
常にクラス受講者一人ひとりの様子をしっかりと見守り続けています。
入門者の不安やストレスをいち早く汲み取り、適切なケアでフォローしてくれるので、安心してヨガに臨めます。
高い説明力とコミュニケーション能力から、着実な上達が可能です。

Message

ヨガは、きっと一生の良き友となるはずです。これから始める皆さまと、一歩一歩を共に歩んでいきたいと思います。戸惑いや挫折もあると思いますが、しっかりとサポートをさせていただきます。一緒に、ゆっくりとヨガの奥深い世界を体感していきましょう。

Class

ビギナークラス、マタニティヨガ、アロマヨガ

戻る

Lotus

Profile

ヨガ歴15年のベテランインストラクターであるLotus。
きびしくも暖かな指導スタイルが高く評価されており、一人ひとりに合わせた的確なアドバイスでクラス受講者から厚い信頼を得ています。
アーサナの徹底したフォロー、呼吸法や瞑想のエッセンスを伝授することで、気付きと癒やしの時間を提供し続けています。
ハイレベルなテクニックと、豊富な知識に裏打ちされた指導力は卓越しており、上級者からも確かな実力を認められています。

Message

心身の調和を保つことで、本来の自分自身に出会うことができます。ヨガにはそんな魅力がございます。一人ひとりの個性と可能性に合わせたクラスを通して、皆さまにヨガの素晴らしさを感じていただきたいと心から願っております。

Class

アシュタンガヨガ、ビンヤサヨガ、ホットヨガ

Sakura

Profile

マタニティヨガの第一人者として広く知られるSakura。
妊娠中のママたちの不安を、優しく丁寧に解消してくれる頼れる存在です。
アーサナや呼吸法、瞑想を通して、ママの心身の変化に寄り添いながら、ママとベビーの絆を一層深められるクラスを提供しています。
安全で効果的なプログラムはママたちから高い支持を得ており、陣痛育児などの知識も兼ね備えており、安心してレッスンを受けられます。

Message

妊娠・出産は、ママになるための望まれし喜びの瞬間です。ヨガが、この大切な時期に寄り添い、サポートできることを心から嬉しく思います。レッスンを通して、安心で健やかな妊娠、産後生活をお手伝いさせていただきます。

Class

マタニティヨガ、ママヨガ、産後ヨガ

戻る

Ken

Profile

豊富な経験とアプローチの多様性から、年齢や性別を問わず幅広い層に人気のKen。
初心者からベテランまで、ニーズに合わせてレッスンを構成します。
ホットヨガでは太陽のエネルギーを取り入れ、メンズヨガでは男性の体の特性を活かしたアーサナを提供します。
いずれのクラスでも丁寧な指導で、ヨガの本質を伝授します。

Message

初めてヨガに挑戦される方も、経験者の方も、ヨガを通して身体を鍛え、心を整えることで、生まれ変わったような開放感を味わえることでしょう。ヨガの奥深い魅力に触れながら、新しい自分との出会いを体感していただければと願っています。

Class

メンズヨガ、ホットヨガ、ビギナークラス

Grace

Profile

ヨガの発祥の地であるインドで長年修業を重ね、正統なシヴァナンダ流ヨガを体現するGrace。
呼吸法とアーサナの調和を重視し、氏の指導を受けると瞑想の境地に至るまでの至高の体験ができます。
現地のヨガの権威から祝福を受けた、まさにヨガの伝道師です。
クラスでは厳格な修行に裏付けられた深遠なヨガの知恵を伝授してくれます。

Message

古来から伝わるシヴァナンダ流ヨガの奥深いエッセンスを、皆さまにしっかりとお伝えできることを、心より楽しみにしております。呼吸に意識を向けることから始まる、この正統なヨガの素晴らしさに触れていただければ幸いです。

Class

シヴァナンダヨガ、アシュタンガヨガ、ビンヤサヨガ
